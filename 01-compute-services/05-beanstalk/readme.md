## bean-stalk

### bean stack provide those features

- **Load Balancing**
- **Auto Scaling**
- **deployment management**
- **rolling updates**
- **monitoring**
- **database management**

## Beanstalk is free, Only pay for the underlying resources (EC2, S3, RDS, etc) that your application uses.

- CloudFormation is used to provision and manage the underlying resources for beanstalk applications.
