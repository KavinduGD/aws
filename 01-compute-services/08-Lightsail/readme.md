# AWS Lightsail

- Simpler alternative to EC2, ESC, EKS , ELB, RDS, Security Groups, and more.
- **Great For Beginners**: Easy to set up and manage, ideal for small projects and learning.
- **Cost-Effective**: Predictable pricing with no hidden fees, making it budget-friendly for small applications.
- You get "Bundles" that include a fixed amount of RAM, vCPU, SSD storage, and a data transfer allowance
- One-click deployments for stacks like MERN, LAMP, Node.js, or apps like WordPress and Ghost.
- Simplified MySQL or PostgreSQL instances that handle backups and scaling independently of your server.
